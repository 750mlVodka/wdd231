Videogame Explorer - Site Plan

By Yamsir Hernandez for WDD 231

Site Name

Videogame Explorer

Domain

Site Purpose

The purpose of this website is to allow users to explore, search, and discover video games using real-time information from the RAWG API. Users can view trending games, read summaries, and create a wishlist of their favorites.

Scenarios

  • Which are the most popular or trending games right now?
  • How can I save my favorite games to a wishlist for later viewing?
  • Can I search for a specific game by name or genre?

Color Schema

Chosen for a dark, modern, and contrast aesthetic:

  • Primary: #10181F - main background
  • Accent: #1DE9B6 - buttons, highlights, links
  • Text: #F2F4F7 - primary text color
  • Light Background: #1B2E35 - cards and sections

Typography

Roboto Mono: simple and slightly technical look, fitting the gaming theme.

Wireframes general structure of the home page for mobile and desktop.
